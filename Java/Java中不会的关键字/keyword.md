# Java中不会的关键字

## Java基础

* 实现接口   implements
* 继承类     extends
* 保护的     protected
* 集合       Collection
* 异常      Exception
* 空指针异常   NullPointerException
* 数组越界异常   IndexOutOfBoundsException
* 还有各种异常
* finally
* 线程     Thread
* 锁      synchronized
* 实现线程的接口    Runnable
* 优先级      Priority
* i/o流  字节的输入输出     InputStream 和 OutputStream
* 比较       equals
* 日期类     Calendar
